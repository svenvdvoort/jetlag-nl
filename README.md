# Jet Lag NL

This is a staticly served webapp to support playing the game 'Jet Lag: Schengen Showdown' on the Dutch rail network, focussed on the Randstad.

The webapp allows teams to track their progress during the game day. It displays claimed and locked cities in a map and a list view, tracks the scores and shows a timer until the end of the game.

The complete webapp was build using the Claude Code free tier.
